# Agents Directory

This directory contains project agents and helper scripts. Structure:

- `agents/setup-client-env/` — setup-client-env agent and installation scripts
- `agents/scrum-master/` — Scrum Master agent documentation
- `agents/developer/` — Developer agent documentation and test result records
- `agents/reviewer/` — Reviewer agent documentation and review result records
- `backlogs/` — product backlog 및 sprint backlog 저장소

Place any new agent under `agents/` as its own folder with an agent.md and related scripts or data.
